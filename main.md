Web开发相关资源、网站、工具推荐
=====

写在前面
-----

请善于使用Ctrl+F以快速获取所需内容。

PS. 由于有些网站的名称不确定，暂时使用页面标题或者页面第一行字代替。

文档
-----

*   [W3C](http://www.w3.org/TR/) (官方文档)
*   [WhatWG](http://www.whatwg.org/) (官方文档)
*   [MDN](https://developer.mozilla.org/) (Mozilla Developer Network, 火狐的文档)  
    // 建议使用英文版，中文版内容不全且更新日期较旧。
